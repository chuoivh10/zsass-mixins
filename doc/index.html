    z-rem

    @mixin z-rem($property, $values, $default-font-size: 16px) { ... }
    View source

    Description

    This mixin enables you to use the CSS3 value rem, which lets you define property sizes based on the root element's font-size.

    Outputs properties that use rem with a px fallback. It also takes px values and converts them to rem.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $property— noneString— none
    $values— noneList— none
    $default-font-size

    optional

    		Number16px

    Example

    .selector {
  @include z-rem(font-size, 1.3);
  @include z-rem(padding, 20px);
}

    Output

      <property>: <parsed value>px;

    z-linear-gradient

    @mixin z-linear-gradient($direction: to bottom, $fallback: #ccc, $from: #ccc, $to: #aaa) { ... }
    View source

    Description

    Generates a linear gradient for a given element with a fallback color.

    Note: By default this linear-gradient-mixin encourages people to use the latest CSS-syntax for gradients.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $direction

    Either an angle value or any of to bottom, to right, to top or to left

    		String or Angleto bottom
    $fallback— noneColor#ccc
    $from— noneColor#ccc
    $to— noneColor#aaa

    Example

    .selector {
  @include x-linear-gradient('to bottom', #ccc, #ddd, #bbb);
}

    Output

      background-color: <fallback>;
  background-image: -webkit-gradient(linear, <direction - old converted>, from(<from>), to(<to>));
  background-image: -webkit-linear-gradient(<direction - converted>, <from>, <to>);
  background-image:         linear-gradient(<direction>, <from>, <to>);

    Requires

    • [function] helper-gradient-angle

    z-multiple-colored-gradient

    @mixin z-multiple-colored-gradient($args...) { ... }
    View source

    Description

    This mixin creates (endless) multiple color stops in gradients just with one call for the mixin.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $args— noneArglist— none

    Example

    .selector {
  @include z-multiple-colored-gradient(
      'top',
      #f22 0%,
      #f2f 15%,
      #22f 30%,
      #2ff 45%,
      #2f2 60%,
      #2f2 75%,
      #ff2 90%,
      #f22 100%
  );
}

    Output

      background-image: -webkit-linear-gradient(top, <stops[1]>, <stops[2]>, ..., <stops[n]>);
  background-image:         linear-gradient(to bottom, <stops[1]>, <stops[2]>, ..., <stops[n]>);
